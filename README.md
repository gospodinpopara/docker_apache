# About

Simple docker configuration with 
- Apache
- PHP 8.3
- Composer 2.4
- Mysql8
- PhpMyAdmin
- Node 16

# Requirements

- docker (26.1)
- docker-compose (2.27)

# Installation

Clone repository
```bash
git clone <repo>
```

Build docker containers
```bash
docker compose build
```

start container
```bash
docker compose up -d 
```
